# AI Automation Course — structură fișiere

Acest folder este o structură de lucru pentru proiectul cursului/webinarului despre AI, automatizări, agenți și Claude/Claude Code pentru persoane non-tehnice.

## Cum se folosește

- `01_docs_readable/` — documente `.docx` pentru citit, discutat și revizuit cu oameni.
- `02_ai_markdown/` — fișiere `.md` scurte și structurate, ideale pentru ChatGPT/Claude/Claude Code.
- `03_research_backlog/` — întrebări, ipoteze și research care trebuie făcut în continuare.
- `04_marketing/` — hook-uri, reclame, funnel și mesaje pentru webinar.
- `05_course_structure/` — structură de curs premium și module.
- `00_admin/` — decizii, changelog și reguli de organizare.

## Principiul de lucru

Nu încercăm să vindem același mesaj tuturor. Avem un produs umbrelă, dar intrări diferite în funnel:

1. Proprietari de afaceri care vor AI și automatizări pentru businessul lor.
2. Profesioniști care vor să devină mai eficienți la job.
3. Freelanceri/consultanți care vor să ofere servicii de AI automation altor afaceri.

## Recomandare scurtă de organizare

Cel mai bun setup este hibrid:

- Google Drive pentru documente citibile, feedback, materiale video și assets.
- Git/GitHub privat pentru `.md`, prompturi, structură de curs, versiuni și lucru cu AI/Claude Code.

Dacă proiectul este la început, începe cu Google Drive + acest folder. Când începi să iterezi serios pe curriculum, reclame, prompturi și materiale AI, mută partea `.md` într-un repo Git privat.
